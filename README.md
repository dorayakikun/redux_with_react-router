## About

redux x react-router のサンプルです。

以下の `npm scripts` を用意しています。

| スクリプト名 | 説明                                                        |
| ------------ | ----------------------------------------------------------- |
| build        | リリース用のバンドルされた JS ファイルを出力します          |
| dev          | webpack-dev-server を起動してサンプルプログラムを実行します |
| lint         | eslint を実行します                                         |
